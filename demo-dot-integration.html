🚛 Vicidial DOT Lookup Integration Demo

📋 What This Integration Does:

  • ✅ Automatically detects when Vicidial leads are imported
  • ✅ Scans for leads with DOT numbers
  • ✅ Queries 428K+ record DB-V3 database automatically
  • ✅ Populates company data, contact info, and business details
  • ✅ Adds vehicle and trailer information from inspection data
  • ✅ Calculates insurance renewal dates and commodity types

🔄 Integration Flow:

1. Import Detection: vicidial-import-listener.js detects new imports
2. DOT Scanning: Scans localStorage for leads with DOT numbers
3. DB-V3 Query: Queries database using /api/test-db/{dotNumber}
4. Data Population: Updates lead profile with carrier information
5. Vehicle Processing: Adds trucks/trailers from inspection data
